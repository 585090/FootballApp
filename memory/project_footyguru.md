FootyGuru is a football match-prediction app: users predict scores, join groups, and compete on leaderboards. The user (Simen) is the sole developer — this is a personal project, not a work product.

**Why:** The user is reviving the project for FIFA World Cup 2026 (June 2026). It sat untouched for ~6 months prior to May 2026.

**How to apply:** Treat this as a solo project with tight timeline pressure — favor pragmatic, ship-it choices over architectural purity. The user is okay with leaving the existing web client in place rather than rewriting everything at once. See [[project-rn-migration]] for the current revival effort.
